# Dash Dashboard: Automobile Recession Analysis

This Python script (Recession_Analysis_automobiledata.py) creates an interactive Dash dashboard that visualizes automobile sales data and highlights trends during recession periods.

## How to Run

1. Make sure you have Python installed (3.8+ recommended).
2. Install the required libraries:

pip install dash pandas plotly

3. Run the Dash app:

python Recession_Analysis_automobiledata.py

4. Open your browser and go to:

http://127.0.0.1:8050

The dashboard will display interactive graphs for automobile sales, vehicle types, and recession periods.
